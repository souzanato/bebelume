# Bloco Sobre Nós

Bloco customizado para seção "Sobre Nós" com design orgânico.

## Características

- Layout em 2 colunas (desktop)
- Background com imagem personalizada
- Título editável
- Conteúdo com múltiplos parágrafos
- Botão configurável (texto e URL)
- Responsivo (mobile sem background)

## Campos Editáveis

- **Título**: RichText editável diretamente no bloco
- **Conteúdo**: RichText com suporte a múltiplos parágrafos
- **Texto do Botão**: Configurável na sidebar
- **URL do Botão**: Configurável na sidebar

## Estrutura de Arquivos

- `block.json` - Configuração do bloco
- `index.js` - Entry point
- `edit.js` - Componente do editor
- `save.js` - Renderização front-end
- `style.css` - Estilos front-end
- `editor.css` - Estilos do editor

## Assets

O bloco utiliza a imagem `assets/images/sobre-nos.png` como background.
